Recherche de Tastee

En tant qu'utilisateur ayant la nécessité d'automatiser des tests
Je veux pouvoir rechercher Tastee dans Google
Afin de pouvoir trouver cet outil rapidement

Recherche via l'URL

Important : si on connait lURL de Tastee, il doit être le 1er lien retourné par Google
Scénario :
//savor rechercher.properties
//savor ../../common/common_fr.yaml
Allez sur le site Google.fr
Renseigner champ.de.recherche avec 'https://tastee.github.io/'
Cliquer sur le boutton Rechercher.bouton
Vérifier que premier.resultat est identique à 'Tastee — test framework : test made Natural'
